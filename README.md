# MySQL 学习笔记
包含学生表、员工/部门表的 DDL（建表）、DML（增删改查）及 15 道 SQL 查询题解答。

## 目录说明
- `student-table/`：学生表相关 SQL（创建、插入、各类查询）
- `emp-dept-table/`：员工/部门/工资等级表相关 SQL + 15 道查询题解答

## 数据库说明
- mydb1：存储学生表（student/student1/student2/student3）
- mybd2：存储部门表（dept1）、员工表（emp1）、工资等级表（salgrade1）

## 核心内容
1. 基础 DDL：创建数据库/表、修改表结构、删除表
2. 基础 DML：插入/删除/更新数据
3. 高级查询：聚合函数、分组、排序、连接查询、子查询
4. 15 道经典 SQL 练习题（员工/部门表）
